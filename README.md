# ⚡ TANZI: High-Performance Productivity & Time Management 🚀

TANZI is a state-of-the-art productivity ecosystem designed for high achievers. It combines elite task tracking, deep focus tools, and advanced analytics into a seamless, cross-platform experience.

---

## 📌 Project Overview

- **Web Portal**: Built with React 18, Vite, and Tailwind CSS.
- **Mobile Support**: Powered by Capacitor v8 (Android/iOS).
- **Backend**: Fully integrated with Firebase (Auth, Firestore, Cloud Functions).
- **Automation**: Intelligent sleep routine management and background check-in reminders.
- **Visuals**: Premium Bento-grid dashboards and real-time Recharts analytics.

## 🚀 Quick Start

1. **Clone & Install**:
   ```bash
   git clone https://github.com/Priyan-19/TANZI.git
   cd TANZI/web
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Explore**:
   Navigate to `http://localhost:5173` to see the live dashboard.
